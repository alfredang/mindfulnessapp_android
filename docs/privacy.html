Privacy Policy — Tertiary Mindfulness

Last updated: 20 June 2026

Tertiary Mindfulness (package com.alfredang.mindfulnesspractice) is a guided-meditation app published by Tertiary Infotech Academy Pte Ltd. Your privacy matters to us, and this app is built to need none of your data.

Data we collect

None. The app has no account system, no analytics, no advertising, and makes no network requests. It does not collect, store, or transmit any personal or device information.

Audio and your music library

The guided meditation voice is bundled inside the app and plays entirely on your device. If you choose to add background music, the app opens your device's system file picker so you can select an audio file. That file is played locally only — it is never uploaded, copied, or shared. The app reads only the single file you explicitly choose.

Permissions

The app requests no sensitive runtime permissions. Background music is accessed through the Android Storage Access Framework, which grants read access only to the file you pick.

Children

The app contains no objectionable content and collects no data, so it is safe for users of all ages. We do not knowingly collect any information from anyone, including children.

Changes

If this policy changes, the updated version will be posted at this URL with a new date above.
